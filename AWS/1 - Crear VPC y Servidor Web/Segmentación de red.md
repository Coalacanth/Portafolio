La segmentación de red está dada por el documento RFC 1918 para los bloques CIDR a las siguientes direcciones de red privada: 
- 10.0.0.0/16
- 172.31.0.0/16
- 192.168.0.0/16
El menor tamaño del bloque permitido es /28, y el mayor es /16.
